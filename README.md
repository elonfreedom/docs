## 开发工具

### Visual Studio Code
- **特点**: 轻量级、高度可定制、丰富的插件生态系统
- **作用**: 代码编辑器，支持多种编程语言，适合前端、后端开发
- **平台**: Windows, macOS, Linux

### JetBrains IntelliJ IDEA
- **特点**: 强大的代码分析和重构工具、智能代码补全
- **作用**: Java 开发的集成开发环境 (IDE)，也支持 Kotlin、Scala 等语言
- **平台**: Windows, macOS, Linux

### Git
- **特点**: 分布式版本控制、高效的分支管理
- **作用**: 版本控制系统，用于跟踪代码变更和团队协作
- **平台**: Windows, macOS, Linux

### Docker
- **特点**: 容器化技术、轻量级虚拟化、易于部署
- **作用**: 应用容器引擎，用于打包、分发和运行应用程序
- **平台**: Windows, macOS, Linux

### Postman
- **特点**: 友好的图形界面、支持自动化测试
- **作用**: API 开发和测试工具，用于发送 HTTP 请求和测试接口
- **平台**: Windows, macOS, Linux

## 办公软件

### Microsoft Office
- **特点**: 功能全面、兼容性强、行业标准
- **作用**: 办公套件，包括 Word、Excel、PowerPoint 等
- **平台**: Windows, macOS

### WPS Office
- **特点**: 免费版可用、体积小、界面友好
- **作用**: 办公软件套件，与 Microsoft Office 高度兼容
- **平台**: Windows, macOS, Linux, iOS, Android

### Notion
- **特点**: 多功能集成、支持数据库和协作
- **作用**: 笔记和知识管理工具，支持团队协作
- **平台**: Windows, macOS, Web, iOS, Android

### Obsidian
- **特点**: 本地存储、支持 Markdown、强大的链接功能
- **作用**: 知识管理和笔记工具，适合构建个人知识库
- **平台**: Windows, macOS, Linux, iOS, Android

## 浏览器

### Google Chrome
- **特点**: 速度快、扩展丰富、跨平台同步
- **作用**: 网页浏览器，开发者工具强大
- **平台**: Windows, macOS, Linux, iOS, Android

### Mozilla Firefox
- **特点**: 开源、注重隐私保护、高度可定制
- **作用**: 网页浏览器，适合注重隐私的用户
- **平台**: Windows, macOS, Linux, iOS, Android

### Microsoft Edge
- **特点**: 基于 Chromium、与 Windows 深度集成、省电
- **作用**: 网页浏览器，Windows 系统默认浏览器
- **平台**: Windows, macOS, Linux, iOS, Android

## 图像处理

### Adobe Photoshop
- **特点**: 专业级图像处理、功能强大、行业标准
- **作用**: 图像编辑软件，适合专业设计师和摄影师
- **平台**: Windows, macOS

### GIMP
- **特点**: 免费开源、功能丰富
- **作用**: 图像编辑软件，Photoshop 的免费替代品
- **平台**: Windows, macOS, Linux

### Figma
- **特点**: 基于浏览器、实时协作、组件化设计
- **作用**: UI/UX 设计工具，适合团队协作
- **平台**: Web, Windows, macOS

### Canva
- **特点**: 易上手、模板丰富、适合非专业人士
- **作用**: 在线图形设计工具，用于制作海报、社交媒体图片等
- **平台**: Web, iOS, Android

## 视频编辑

### Adobe Premiere Pro
- **特点**: 专业级视频编辑、功能全面、行业标准
- **作用**: 视频编辑软件，适合专业视频制作
- **平台**: Windows, macOS

### Final Cut Pro
- **特点**: 专为 Mac 优化、流畅的操作体验
- **作用**: 专业视频编辑软件，Mac 平台首选
- **平台**: macOS

### DaVinci Resolve
- **特点**: 免费版功能强大、专业调色工具
- **作用**: 视频编辑和调色软件，适合专业用户
- **平台**: Windows, macOS, Linux

### 剪映
- **特点**: 简单易用、模板丰富、智能功能多
- **作用**: 视频编辑软件，适合短视频制作
- **平台**: Windows, macOS, iOS, Android

## 系统工具

### CCleaner
- **特点**: 清理垃圾文件、优化注册表、管理启动项
- **作用**: 系统清理和优化工具
- **平台**: Windows, macOS

### 7-Zip
- **特点**: 开源免费、支持多种压缩格式、压缩率高
- **作用**: 文件压缩和解压工具
- **平台**: Windows, Linux

### VLC Media Player
- **特点**: 开源免费、支持几乎所有视频格式、无需额外编解码器
- **作用**: 多媒体播放器
- **平台**: Windows, macOS, Linux, iOS, Android

### Everything
- **特点**: 极速搜索、占用资源少、实时更新
- **作用**: 文件搜索工具（Windows）
- **平台**: Windows

## 通讯工具

### Slack
- **特点**: 频道化沟通、丰富的集成、支持远程协作
- **作用**: 团队沟通和协作工具
- **平台**: Windows, macOS, Linux, Web, iOS, Android

### Microsoft Teams
- **特点**: 与 Office 365 集成、视频会议、团队协作
- **作用**: 企业通讯和协作平台
- **平台**: Windows, macOS, Linux, Web, iOS, Android

### Zoom
- **特点**: 高质量视频会议、易于使用、稳定性好
- **作用**: 视频会议软件
- **平台**: Windows, macOS, Linux, iOS, Android

### Discord
- **特点**: 低延迟语音、社区功能强大、免费
- **作用**: 语音、视频和文字聊天工具，最初面向游戏玩家
- **平台**: Windows, macOS, Linux, Web, iOS, Android

## 云存储

### Google Drive
- **特点**: 15GB 免费空间、与 Google 服务集成、实时协作
- **作用**: 云存储和文件共享服务
- **平台**: Web, Windows, macOS, iOS, Android

### Dropbox
- **特点**: 自动同步、版本历史、简单易用
- **作用**: 云存储和文件同步服务
- **平台**: Windows, macOS, Linux, Web, iOS, Android

### OneDrive
- **特点**: 与 Windows 深度集成、Office 文件协作
- **作用**: 微软云存储服务
- **平台**: Windows, macOS, Web, iOS, Android

### 百度网盘
- **特点**: 大容量免费空间、国内访问速度快
- **作用**: 云存储服务，适合国内用户
- **平台**: Windows, macOS, Web, iOS, Android
